## 🧠 Why TypeScript?

TypeScript is a **strongly typed superset of JavaScript** that helps developers build safer and scalable applications.

Benefits include:

- Static typing
- Better IDE support
- Early error detection
- Safer refactoring
- Better large-scale code maintenance

Companies using TypeScript include:

- Microsoft
- Airbnb
- Slack
- Shopify
- Uber
